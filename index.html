Medal Clip Webhook

Being a Medal user, I've seen some Guilded or even Slack users want Medal clip sharing for their platform.
While using it directly from the app is easier, I decided to make an app that can send a webhook message to either platform with a clip link.
This can be Self Hosted, but you'll need some HTML and Javascript experience most likely (depending on what you want to do with it).
